# Udacity Blockchain Capstone

This is a completed Capstone project, as part of the Udacity assignment.

# Install
```
npm cache clean --force
rm -rf node_module
rm -rf eth-contracts/build
npm install
```

# Testing

There are three tests within eth-contracts\test, that tests three smart contracts. `TestERC721Mintable.js` is used to test against the customer ERC721 implementation. `TestSolnSquarVerifier.js` is used to test the complete zksnarks along with the ERC721 minting. `TestSquareVerifier` is dedicated to test the zksnarks from the generated proof only.

To run tests:

```
cd eth-contracts
truffle dev
test
```

# Deploy

Make sure you create a local file `.env` within eth-contracts, with the format:

API_KEY="{APY_KEY}"
MNEMONIC="{MNEMONIC}"
NFT_CONTRACT_ADDRESS="{NFT_CONTRACT_ADDRESS}"
OWNER_ADDRESS="{OWNER_ADDRESS}"


Then you can run the command without erroring out:

```
cd eth-contracts
truffle migrate --network rinkeby --reset
```

# Mint

After deploying smart contract, we need to mint against it so that the new ERC 721 tokens will be minted. To do so:

```
cd eth-contracts
node mint.js
```
